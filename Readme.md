# PsSqlTestTasks

[PsBuildTasks](https://github.com/abbgrade/PsBuildTasks) extension to test tools for SQL Server and Azure SQL.

## Installation

```powershell
Install-PsBuildExtension `
    -Path $PSScriptRoot/.. `
    -Name PsSqlTestTasks `
    -Repository 'https://github.com/abbgrade/PsSqlTestTasks.git'
```
